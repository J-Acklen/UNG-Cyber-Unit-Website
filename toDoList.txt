================================================================================
  CYBERSECURITY BASICS — NPM WEB APP
  To-Do List for Claude Code
  Audience: Cybersecurity beginners / college intro class
  Stack: Node.js + Express (backend) + Vanilla HTML/CSS/JS (frontend)
================================================================================

GOAL:
  Build a multi-page, interactive educational web app that teaches the absolute
  basics of cybersecurity to complete beginners. The app should feel like a
  polished "field guide" — dark terminal aesthetic, hacker-green accents,
  monospace typography, clean layout. Every section should teach one concept
  with a short explanation, a visual/diagram, and a hands-on mini-demo or quiz.
